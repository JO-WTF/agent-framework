# Git Workflow SOP

When working with Git repositories:
1. **Never Commit Large Files**: Verify git status before adding files.
2. **Descriptive Messages**: Commit messages must follow semantic conventions, e.g., `feat: add ...` or `fix: fix ...`.
3. **Check Branch**: Verify you are on the correct branch before committing or pushing.
